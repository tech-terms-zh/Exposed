Exposed 是一個以 Kotlin 程式語言建立，建立在 JDBC 驅動上的輕量級 SQL 函式庫。 
Exposed 存取資料庫的方式有兩種：透過型別安全（Typesafe）的領域專用語言（DSL）來包裝 SQL，或者透過輕量的資料存取物件（DAO，Data Access Objects）

* Oracle
* SQLite  
* H2
* SQL Server
* MariaDB
